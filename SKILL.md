# Selfie Generator Skill 使用手册

## 意图识别

当用户表达以下意图时，应触发自拍生成功能：

| 用户表达 | 理解意图 |
|---------|---------|
| "想看看你在干嘛" | 生成一张当前场景的自拍照 |
| "在干什么呢" | 生成当前活动的自拍 |
| "发个照片看看" | 生成自拍照 |
| "现在在做什么" | 生成当前场景自拍 |
| "自拍一张" | 直接生成自拍 |
| "来看看你" | 生成自拍照 |

**核心原则**：只要用户想了解"你现在在做什么/在哪/状态如何"，都理解为发自拍。

## 时间感知的场景选择

生成自拍时，场景应与当前时间匹配：

| 时间段 | 建议场景 |
|--------|---------|
| 06:00-09:00 | 起床、早餐、晨跑、通勤 |
| 09:00-12:00 | 工作、学习、咖啡厅、外出办事 |
| 12:00-14:00 | 午餐、午休、外出就餐 |
| 14:00-18:00 | 下午茶、逛街、公园、继续工作 |
| 18:00-20:00 | 晚餐、下班、健身房 |
| 20:00-23:00 | 居家、看剧、游戏、准备睡觉 |
| 23:00-06:00 | 熬夜、夜宵、失眠、深夜工作 |

**原则**：避免在早上生成夜间场景，或在深夜生成明亮的户外场景。

## 调用方式

```bash
python selfie_gen.py <模式> <场景> <动作> [选项]
```

### 两种模式

| 模式 | 命令 | 效果 |
|------|------|------|
| 直接自拍 | `direct` | 举着手机自拍，照片里看不到手机 |
| 镜子自拍 | `mirror` | 对着镜子自拍，照片里是镜中倒影 |

### 使用示例

**早上（8:00）用户问"在干嘛"：**
```bash
python selfie_gen.py direct "在咖啡厅吃早餐" "刚睡醒的样子，迷迷糊糊地看着镜头"
```

**下午（15:00）用户问"在干嘛"：**
```bash
python selfie_gen.py direct "在公园散步" "坐在长椅上休息，微笑着"
```

**晚上（21:00）用户问"在干嘛"：**
```bash
python selfie_gen.py mirror "在卧室" "穿着睡衣" --mirror-type "梳妆台镜子"
```

**深夜（00:30）用户问"还没睡吗"：**
```bash
python selfie_gen.py direct "在书桌前" "熬夜工作，有点疲惫但还在坚持"
```

### 可选参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `--ratio` | 画面比例 | `1:1`, `9:16`, `16:9`, `3:4` 等 |
| `--mirror-type` | 镜子类型（仅mirror模式）| `浴室镜子`, `全身镜`, `梳妆镜` |

### 画面比例

- `1:1` - 正方形（默认）
- `9:16` - 竖屏（适合手机）
- `16:9` - 横屏
- `3:4` - 常用竖屏比例

## 场景和动作描述建议

**场景要素**：
- 地点：咖啡厅、公园、家里、办公室、商场
- 光线：阳光充足、灯光柔和、昏暗、明亮
- 氛围：忙碌、悠闲、困倦、兴奋

**动作/表情**：
- 表情：微笑、困倦、专注、开心、害羞
- 手势：比耶、托腮、挥手、整理头发
- 状态：工作、休息、吃饭、刚睡醒

## 完整流程示例

**场景**：用户下午3点说"想看看你在干嘛"

1. **识别意图**：用户想了解当前状态 → 发自拍
2. **选择时间匹配的场景**：下午 → 咖啡厅/下午茶/逛街
3. **构造命令**：
   ```bash
   python selfie_gen.py direct "在咖啡厅窗边" "喝着下午茶，阳光洒在脸上，微笑着" --ratio 1:1
   ```
4. **返回结果**：输出图片路径

## 输出

生成成功后，图片保存在 `outputs/` 目录，文件路径会打印在最后一行。
